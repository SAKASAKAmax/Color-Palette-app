# Color Palette Generator

このプロジェクトは、プレゼン資料やWebデザイン向けの美しいカラーパレットを生成するWebサービスです。

## デプロイ方法（Vercel）

1. [Vercel](https://vercel.com/) にサインアップし、このリポジトリをインポート
2. デプロイボタンを押すだけで公開できます

- 静的なHTML/CSS/JSのみで動作します
- `index.html`がトップページとして公開されます

## ディレクトリ構成

```
Color_Palette/
├── index.html
├── src/
│   ├── main.js
│   └── style.css
```
